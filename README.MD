Constellation-16
===

Announced in 2012, `0x10c` was going to be a science-fiction game with one stand-out feature: A fully-simulated 16-bit processor that the players would have to use in order to power the systems aboard their ships.

The game, unfortunately, was canceled in 2013. Thankfully, the specs for the fantasy-CPU [were preserved](https://github.com/lucaspiller/dcpu-specifications).

I thought it would be fun to try and implement them, here's hoping the project doesn't prove me wrong on that.

Note: **constellation-16** is the overarching project, while the **PerCPU** refers to only the CPU itself.
